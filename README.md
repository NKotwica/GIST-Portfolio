# GIST-Portfolio

This repository contains GIS labs and final projects completed as part of GEOG 411 and GEOG 413. The work focuses on applying spatial analysis and remote sensing techniques to real-world problems.

## Key Features
- Hands-on GIS labs and two comprehensive final projects  
- Workflows developed in Google Colab using Python  
- Integration of Google Earth Engine for remote sensing and large-scale spatial analysis  
- Emphasis on data visualization and clear communication of spatial insights  

## Skills Demonstrated
- Spatial analysis and geoprocessing  
- Remote sensing and Earth observation  
- Data visualization and mapping  
- Applied GIS problem solving
